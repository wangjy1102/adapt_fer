#  ADAPT-Pytorch
ADAPT: A Self-Evolving Meta-Learning Method for Classroom Facial Expression Recognition

This repository contains the PyTorch imploymentation for the ADAPT. If you have any questions on this repository or related paper, feel free to create an issue or send me an email.

Migrate code to the following environment:

Python 3.10

Torch 2.0.0

numpy 2.0.2

Scikit-learn 1.3.2

#### pip

Write the command below in your terminal to install the dependencies according to the configuration file requirements.txt.

```
pip install -r requirements.txt
```
